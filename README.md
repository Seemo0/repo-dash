# Repository Dashboard

Next.js front-end application to browse code repositories.

## Requirements

- Node.js 18 or later
- npm

## Installation

```bash
git clone https://github.com/Seemo0/repo-dash.git
```

```bash
cd repo-dash
```

```bash
npm install
```

## Development

```bash
npm run dev
```

Open http://localhost:3000 in your browser.
